tech.v3.dataset.tensor

Conversion mechanisms from dataset to tensor and back as will as zerocopy bindings to smile dense matrixes. Smile has full BLAS/LAPACK bindings so this makes it easy to move from a dataset into transformations such as PCA and back. In order to use pca you need to add either latest javacpp openblas support smile mkl support to your project:
