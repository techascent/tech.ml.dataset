input->workbook

(input->workbook input)(input->workbook input options)

Given an input data source, return an implementation of tech.v3.dataset/Spreadsheet$Workbook. This interface allows you to iterate through sheets without necessarily parsing them. Once you have a spreadsheet, use tech.v3.dataset.io.spreadsheet/sheet->dataset to get a dataset.

workbook->datasets

(workbook->datasets input options)(workbook->datasets workbook)

Returns a sequence of dataset named after the sheets. This supports a subset of the arguments for tech.v3.dataset/->dataset. Specifically:

  • :header-row?
  • :parser-fn
  • :parser-scan-len

Returns a non-lazy sequence of datasets.
